# Contributing to controllercheck.org

Thank you for your interest in contributing to controllercheck.org.  
This project is still in an early stage and any help is greatly appreciated.

---

## How to Contribute

You can help by:

- Reporting bugs
- Suggesting new features
- Improving the code
- Improving documentation
- Testing with different controllers and browsers

---

## Submitting Changes

1. Fork this repository
2. Create a new branch for your change
3. Make your changes
4. Commit with a short, clear message
5. Open a Pull Request

---

## Bug Reports

Please include:

- Browser and version  
- Operating system  
- Controller model  
- Steps to reproduce the issue  

---

## Feature Requests

Feature ideas and improvements are welcome.  
Please describe clearly what you would like to add or change.

---

## Code Style

Keep the code simple, readable, and consistent with the existing style.

---

Thank you for helping to improve this project.
